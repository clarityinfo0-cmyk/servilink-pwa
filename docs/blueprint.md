# **App Name**: ServiLink

## Core Features:

- User Authentication & Profiles: Secure login and registration for Clients, Technicians, and Admins, allowing access to role-specific functionalities.
- Ticket Creation: Clients can easily submit new maintenance requests, including detailed problem descriptions and photo uploads.
- Ticket Dashboard & Details: Consolidated views for Clients to track their requests, Technicians to view assigned tasks, and Admins to oversee all ongoing services.
- Technician Work Management: Technicians can update ticket statuses (e.g., 'assigned', 'in process', 'completed') and upload 'before'/'after' photo evidence directly from their panel.
- Administrator Control Panel: Centralized interface for Admins to view all tickets, assign technicians to tasks, and manage service flows.
- Technician Verification Workflow: A robust process for Admins to review and approve technician applications, including verifying credentials (INE, photo, specialty).
- AI-powered Diagnostic Assistant Tool: A generative AI tool to assist technicians by suggesting potential diagnoses or solutions based on the client's initial problem description, speeding up problem resolution.

## Style Guidelines:

- Primary color: A deep, professional blue (#2B66A1) that conveys trust and efficiency, aligning with a reliable service platform.
- Background color: A very light, desaturated blue (#EDF2F7) to maintain a clean, expansive, and approachable aesthetic for the user interface.
- Accent color: A vibrant cyan (#1AA6B3) used sparingly for key interactive elements like buttons and notifications to provide clear calls to action and visual emphasis.
- Body and headline font: 'Inter' (sans-serif), chosen for its modern, highly readable, and neutral characteristics, suitable for conveying clear information in forms, dashboards, and reports across all user roles.
- Use a set of simple, modern, and intuitive vector icons to represent different service categories, actions, and status updates, ensuring universal understanding.
- Prioritize a mobile-first, responsive design with clear, organized forms and dashboards, enabling efficient use by technicians in the field and administrators on the go.
- Implement subtle and purposeful animations for state changes, loading indicators, and form submissions to enhance perceived responsiveness and provide visual feedback to users.
